# 🌐 Proyecto Web con HTML y CSS

¡Bienvenido a este proyecto! 🚀  
Este es un sitio web desarrollado utilizando **HTML5** y **CSS3**, enfocado en crear una interfaz atractiva, organizada y responsive. 🎨💻

## 📌 Descripción

Este proyecto fue creado con el objetivo de practicar y fortalecer conocimientos en maquetación web, estructura semántica y estilos visuales modernos.  

Está construido únicamente con:

- 🧱 **HTML** para la estructura del contenido
- 🎨 **CSS** para el diseño, colores, tipografías y distribución visual

## 🛠️ Tecnologías utilizadas

- ✅ HTML5
- ✅ CSS3

## ✨ Características

- 📱 Diseño responsive
- 🎯 Estructura semántica
- 🎨 Estilos personalizados
- 🚀 Código limpio y organizado
- 💡 Fácil de entender y modificar
